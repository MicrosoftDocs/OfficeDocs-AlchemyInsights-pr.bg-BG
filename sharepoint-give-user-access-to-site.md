# <a name="give-users-access-to-sharepoint-site"></a>Дават на потребителите достъп до сайта на SharePoint

<p>Функцията за искане на достъп позволява на хората да поискате достъп до съдържание, което те в момента нямат разрешение да виждат. Като собственик на сайт можете да конфигурирате функцията да изпрати поща, когато някой поиска достъп до сайта.
