# <a name="submit-dynamics-365-support-requests"></a>Подаване на заявки за поддръжка на Dynamics 365

Ако сте клиент на **Microsoft Dynamics клиента ангажимент** Моля, изпратете вашия билет чрез новата [мощност платформа администратор център](https://admin.powerplatform.microsoft.com/?ref=officemodern).
  
Новият портал предоставя по-пряк път към подпомага подкрепа, която е оптимизирана за динамиката на клиента ангажимент и други продукти, мощност платформа.
