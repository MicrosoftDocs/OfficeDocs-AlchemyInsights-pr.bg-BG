# <a name="set-field-values-using-parameters-passed-to-a-form---lookup-fields"></a>Задаване на стойности на полетата, като се използват параметри, предадени на формуляр - справочни полета

Задайте стойността за справочни полета и пример: Използвайте Xrm.Utility.openEntityForm, за да отворите нов прозорец. Не можете да зададете стойности за partylist или относно справки.

За повече информация вижте [Задаване на стойности за справочни полета](https://docs.microsoft.com/previous-versions/dynamicscrm-2016/developers-guide/gg334375(v=crm.8)#set-values-for-lookup-fields).
