# <a name="join-a-team-by-code"></a>Присъединете се към екип с код

Microsoft е наясно с проблем, при който потребителите не могат да създадат код за **присъединяване към екип** или да се присъединят към екип с помощта на функцията " **съединение по код** ". Като заобиколно решение, собствениците на екипи могат да добавят ръчно потребители, ако е необходимо.
