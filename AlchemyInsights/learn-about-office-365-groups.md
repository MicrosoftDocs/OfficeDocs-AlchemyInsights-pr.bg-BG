# <a name="learn-about-microsoft-365-groups"></a>Научете за групите на Microsoft 365

Microsoft 365 Groups е услуга, която работи с инструментите за Office 365, които използвате вече, така че можете да си сътрудничите със съотборниците си, когато пишете документи, създавате електронни таблици, работите по планове на проекти, насрочване на събрания или изпращане на имейл. За повече информация вижте [Научете за групите на Microsoft 365](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
