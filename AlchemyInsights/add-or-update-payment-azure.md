# <a name="update-billing-info-under-azure-enterprise-agreement"></a>Актуализиране на информацията за фактуриране под Azure Enterprise споразумение

Промяна на вашата информация за плащане за абонамент за Azure Enterprise не се поддържа.

Ако искате да промените адреса на вашата съществуваща фактура, обърнете се към вашия представител на Microsoft.
