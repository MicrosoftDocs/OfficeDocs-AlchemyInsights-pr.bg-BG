"Неподдържана операционна система"

# <a name="solutions-for-unsupported-operating-system"></a>Решения за "Неподдържана операционна система"

Когато се опитате да инсталирате Office 365/2016, може да видите следното съобщение за грешка, ако вашето устройство не отговаря на [системните изисквания](https://products.office.com/office-system-requirements).
  
 **"Неподдържана операционна система"**
  
Office 365/2016 изисква:
  
PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 или Windows Server 2008 R2
  
или 
  
Mac OS X 10.10
  
Ако имам това съобщение при опит да инсталирате Office на мобилно устройство или Chromebook, вижте [Настройка на Office приложения и имейл на мобилно устройство](https://support.office.com/article/7dabb6cb-0046-40b6-81fe-767e0b1f014f?wt.mc_id=Alchemy_ClientDIA.aspx).
