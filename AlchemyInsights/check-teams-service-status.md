# <a name="check-teams-service-status"></a>Състояние на услугата за проверка на екипи

За да проверите текущото състояние на услугата Microsoft Teams, отидете на Microsoft 365 Център за администриране на здравни табла за услуги.

За повече информация вижте [как да проверите здравето на microsoft 365 услуги](https://docs.microsoft.com/office365/enterprise/view-service-health).
