# <a name="changes-to-external-user-access"></a>Промени във външен потребителски достъп

От 23 март 2018 г. външни потребители няма да имат достъп до съдържание, което е споделено с всички, всички удостоверени потребители или потребители на всички формуляри, дори ако са били поканени или добавени към упълномощени групи. Външните потребители ще имат достъп само до съдържание, което е споделено с потребителя или с групи, към които принадлежи потребителят. Ако искате да запазите съществуващите си настройки, моля, вижте:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
