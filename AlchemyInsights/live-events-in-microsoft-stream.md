# <a name="live-events-in-microsoft-stream"></a>Събития на живо в Microsoft Stream

Можете да създавате събития на живо с помощта на Microsoft Stream в организацията. Можете да планирате, произвеждате и да предоставяте събития на живо за различни сценарии, като например събития в цялата компания, актуализации на лидерството и др.

- [Събития на живо в Microsoft Stream](https://docs.microsoft.com/stream/live-event-overview)
- [Създаване на събитие на живо в Microsoft Stream](https://docs.microsoft.com/stream/live-create-event)
- [Първи стъпки със събитията на живо на Microsoft Teams](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
- [Конфигуриране на кодери за пряко поточно предаване в Microsoft Stream](https://docs.microsoft.com/stream/live-encoder-setup)
