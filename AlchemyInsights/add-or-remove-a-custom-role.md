# <a name="add-or-remove-a-custom-role"></a>Добавяне или премахване на роля по избор

За да управлявате ресурсите на Azure, вижте [Добавяне или премахване на присвоявания на Azure роля с помощта на портала на Azure](https://docs.microsoft.com/azure/role-based-access-control/role-assignments-portal).

За да управлявате роли по избор за Azure Active Directory (Azure AD), вижте [потребителски администраторски роли в Azure Active Directory (предварителен преглед)](https://docs.microsoft.com/azure/active-directory/users-groups-roles/roles-custom-overview).
