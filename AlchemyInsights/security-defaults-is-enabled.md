# <a name="security-defaults-is-enabled"></a>Настройката по подразбиране за защита е разрешена

Настройката по подразбиране на защитата подканва всички потребители за многофакторно удостоверяване.

За да управлявате настройките по подразбиране на защитата:

1. Влезте в портала на [Azure](https://ms.portal.azure.com/)   с вашите идентификационни данни на глобален администратор.
2. Отидете на свойствата на **Azure Active Directory**  >  **Properties**.
3. В долния край на страницата щракнете върху **Управление на настройките по подразбиране за защита**.
4. Щракнете върху **Да**, за да разрешите настройките по подразбиране за защита или върху **Не**, за да забраните настройките по подразбиране за защита.

За повече информация вижте [Какво представляват настройките по подразбиране за защитата?](https://docs.microsoft.com/azure/active-directory/fundamentals/concept-fundamentals-security-defaults).
