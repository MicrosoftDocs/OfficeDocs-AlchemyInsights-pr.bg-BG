# <a name="already-paid"></a>Вече платиха?

Ако вече сте изпратили плащане, можете да ни уведомите:
  
1. В центъра за администриране отидете на страницата [Абонаменти за](https://go.microsoft.com/fwlink/p/?linkid=842054) **фактуриране.** \>

2. Изберете абонамент и в раздела **Моята сметка** изберете Преглед **на бил**.

3. На страницата **С подробности за сметката** изберете **Вече платен?** Ще бъдете помолени да въведете информация, свързана с изпратените плащания.

4. Когато цялата необходима информация е въведена, изберете **Изпращане**.
