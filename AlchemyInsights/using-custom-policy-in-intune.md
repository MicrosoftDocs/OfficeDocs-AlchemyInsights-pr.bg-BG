# <a name="using-custom-policy-in-intune"></a>Използване на правила по избор в настройка

В настройката има много вградени настройки за устройства с Android, iOS, macOS и Windows. Можете също да създадете потребителски профил и след това да разположите тези персонализирани настройки на вашите устройства с помощта на "Настройка". Потребителските профили са предназначени да добавят настройки и функции на устройството, които не са вградени в настройката.

[Настройки по избор за устройства с Android в съзвучие](https://docs.microsoft.com/intune/custom-settings-android)

[Настройки по избор за устройства с Android Enterprise в съзвучие](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Настройки по избор за устройства с iOS в съзвучие](https://docs.microsoft.com/intune/custom-settings-ios)

[Персонализирани настройки за macOS устройства в съзвучие](https://docs.microsoft.com/intune/custom-settings-macos)

[Настройки по избор за устройства с Windows 10 в съзвучие](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Персонализирани настройки за Windows холографски за бизнеса устройства в съзвучие](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
