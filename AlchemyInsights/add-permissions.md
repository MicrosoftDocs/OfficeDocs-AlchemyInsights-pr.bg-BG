# <a name="add-permissions"></a>Добавяне на разрешения

1. Използвайте крайна точка за съгласие на администратор, за да [конфигурирате списъка със статично заявени разрешения за приложение](https://docs.microsoft.com/azure/active-directory/develop/v2-permissions-and-consent#to-configure-the-list-of-statically-requested-permissions-for-an-application).
1. Има две разрешения, налични за даване на възможност за създаване на регистрации на приложения, всеки с различно поведение. За повече информация вижте [разрешения за регистрация на приложения за роли по избор в Azure Active Directory](https://docs.microsoft.com/azure/active-directory/roles/custom-available-permissions).
1. [Разрешения за роли](https://docs.microsoft.com/azure/active-directory/roles/permissions-reference#role-permissions): таблици, които описват конкретните разрешения в Azure Active Directory за всяка роля. Някои роли може да имат допълнителни разрешения в услугите на Microsoft извън Azure Active Directory.
