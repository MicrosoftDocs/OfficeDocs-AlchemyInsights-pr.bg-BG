# <a name="getting-custom-error-module-does-not-recognize-this-error-when-downloading-a-message-trace"></a>Получаване на "потребителски модул за грешка не разпознава тази грешка" при изтегляне на проследяване на съобщение?

Ние сме наясно с проблема и работим върху корекция.  Междувременно опитайте да **изчистите кеша на браузъра**или като използвате режим InPrivate, за да отстраните този проблем.
