# <a name="problem-when-signing-in-to-my-application"></a>Проблем при влизане в моето приложение

Можете да решите проблеми с достъпа до прокси сървъра на приложението, като използвате статиите:

- Получавам съобщение [за грешка "не може да се осъществи достъп до тази корпоративна програма"](https://docs.microsoft.com/azure/active-directory/application-proxy-sign-in-bad-gateway-timeout-error/?WT.mc_id=UI_AAD_Enterprise_Apps_Support_L2_Overview).
- Не [мога да вляза в локално приложение чрез прокси сървъра за приложения на AZURE ad](https://docs.microsoft.com/azure/active-directory/application-sign-in-problem-on-premises-application-proxy/?WT.mc_id=UI_AAD_Apps_Sign_In_Support_L2_Proxy).
- Отстраняване на проблеми с приложението за [прокси сървър на приложения](https://docs.microsoft.com/azure/active-directory/manage-apps/application-proxy-debug-apps).
