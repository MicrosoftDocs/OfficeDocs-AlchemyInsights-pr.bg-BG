# <a name="microsoft-bookings-payment-options"></a>Опции за плащане на резервации на Microsoft

Опциите за плащане на Microsoft Bookings са оттеглени март 2020. Организациите вече използват решения на трети страни, за да добавят опция за плащане към резервации.
