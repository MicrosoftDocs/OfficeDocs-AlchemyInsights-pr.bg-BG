# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Поддържана версия на инструмента за мигриране на SharePoint



В момента Инструментът за мигриране на SharePoint поддържа само миграции от SharePoint 2013. Версии 2010 и 2016 не се поддържат в момента.
  
За повече информация вижте [въвеждане на инструмента за мигриране на SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
