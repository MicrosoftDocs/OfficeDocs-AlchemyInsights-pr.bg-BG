# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Поддържана версия на инструмента за миграция на SharePoint



В момента Инструментът за миграция на SharePoint поддържа само миграции от SharePoint 2013. По това време не се поддържат версии 2010 и 2016.
  
За повече информация вижте [въвеждане на инструмента за миграция на SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
