# <a name="use-microsoft-graph-api-to-manage-groups"></a>Използвайте Microsoft Graph API за управление на групи

Групите са колекции от потребители и други главни лица, които споделят достъпа до ресурси в Microsoft Services или в приложението. Microsoft Graph предоставя API, които можете да използвате, за да създавате и управлявате различните типове групи и функции на групата според вашия сценарий. Всички операции, свързани с групата, в Microsoft Graph изискват одобрение от администратор. За повече информация вижте [работа с групи в Microsoft Graph](https://docs.microsoft.com/graph/api/resources/groups-overview).
