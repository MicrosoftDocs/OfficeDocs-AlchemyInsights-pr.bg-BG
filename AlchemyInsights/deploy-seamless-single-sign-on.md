# <a name="enable-seamless-sso"></a>Давам възможност на Seamless SSO

Лазурно Active Directory (Azure AD) безшевни еднократна идентификация (безшевни SSO) автоматично знаци в потребителите, когато те са на своите корпоративни настолни компютри, които са свързани с вашата корпоративна мрежа.
  
[Следвайте стъпките в тази статия за да активирате безшевни див](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
