# <a name="enable-seamless-sso"></a>Разрешаване на безпроблемен SSO

Azure Active Directory (Azure AD) безшевни еднократна идентификация (безпроблемно SSO) автоматично се подписва при потребители, когато те са на своите корпоративни настолни компютри, които са свързани с вашата корпоративна мрежа.
  
[Следвайте стъпките в тази статия, за да разрешите безпроблемно SSO](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
