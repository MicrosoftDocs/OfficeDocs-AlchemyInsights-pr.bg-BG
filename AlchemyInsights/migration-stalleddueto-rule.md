# <a name="stalleddueto-errors-during-migration"></a>StalledDueTo грешки по време на миграцията

Възможно е да видите редица грешки, които започват с "StalledDueTo..." по време на мигрирането към Exchange Online. Много от тези грешки са напълно нормални. За повече информация вижте [ограничаване на ресурсите и приоритизиране в мигриране на Exchange Online](https://techcommunity.microsoft.com/t5/exchange-team-blog/resource-based-throttling-and-prioritization-in-exchange-online/ba-p/608020).
