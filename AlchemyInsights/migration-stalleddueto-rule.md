# <a name="stalleddueto-errors-during-migration"></a>ЗабавеноDueЗа грешки по време на миграция

Може да видите различни грешки, които започват с "StalledDueTo..." по време на вашата миграция към Exchange Online. Много от тези грешки са напълно нормални. За повече информация вижте [ресурси базирани ограничаване и приоритизиране в Exchange Online миграции](https://techcommunity.microsoft.com/t5/exchange-team-blog/resource-based-throttling-and-prioritization-in-exchange-online/ba-p/608020).
