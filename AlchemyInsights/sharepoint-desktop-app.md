# <a name="desktop-app-for-sharepoint"></a>Настолно приложение за SharePoint

Вашата организация се въвежда в програмата за частен преглед за настолното приложение за SharePoint.

Microsoft в момента не поддържа настолното приложение за SharePoint (частен преглед). Използвайте своите партньори на Microsoft Private визуализация, за да получите помощ с настолното приложение за SharePoint.
