# <a name="update-taxvat-id"></a>Актуализиране на данъчни/ДДС ИД

Ако имате някакви проблеми, свързани с вашия данъчен/ДДС ИД, обърнете се към вашия партньор за лицензиране на Microsoft (партньор или софтуерен съветник). Те ще се свържат с регионален оперативен център (ROC) за въпроси, свързани с вашия данъчен/ДДС ИД.
