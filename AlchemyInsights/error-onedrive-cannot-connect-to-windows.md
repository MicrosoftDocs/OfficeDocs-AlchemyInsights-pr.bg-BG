# <a name="error-onedrive-cannot-connect-to-windows"></a>Грешка: OneDrive не може да се свърже с Windows

За да разрешите съобщението за грешка, потърсете файла на cldflt.sys, за да видите дали работи и ако не е, го разрешете. 
  
За допълнителна информация относно разрешаването на файла cldflt.sys вижте [OneDrive не може да се свърже с Windows" грешка при достъп до файлове в OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
