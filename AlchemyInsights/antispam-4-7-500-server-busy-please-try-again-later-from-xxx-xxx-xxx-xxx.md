# <a name="47500-server-busy-please-try-again-later"></a>4.7.500 сървър зает, моля опитайте отново по-късно

Тази грешка възниква, когато обемът на имейл трафик от IP адреса на източника надвишава ограничението въз основа на репутацията (или липсата на репутация) на източник IP адрес.
  
Блокиране на имейл от IP адреса на източника ще изтече в рамките на един час. Ако изходният IP адрес не е локален имейл сървър, който принадлежи на вас, проверете конфигурацията на на пощенски поток конектор. Ако поведението продължава за повече от един час, се свържете с поддръжката да поискате изключение за IP адреса на източника.
