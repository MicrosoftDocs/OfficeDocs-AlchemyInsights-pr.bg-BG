# <a name="47500-server-busy-please-try-again-later"></a>4.7.500 сървърът е зает, опитайте отново по-късно

Тази грешка възниква, когато обемът на имейл трафик от IP адреса на източника надвишава ограничението въз основа на репутацията (или липсата на репутация) на IP адреса на източника.

Блокирането на имейл от IP адреса на източника ще изтече в рамките на един час. Ако ИЗТОЧНИК IP адрес ът е локален имейл сървър, който принадлежи на вас, проверете конфигурацията на конектор за пощенски поток. Ако поведението продължава повече от час, свържете се с поддръжката да поиска изключение за IP адреса на източника.
