# <a name="47500-server-busy-please-try-again-later"></a>4.7.500 Server е зает, моля, опитайте отново по-късно

Тази грешка се появява, когато обемът на имейл трафика от изходния IP адрес надвишава ограничението на базата на репутацията (или липсата на репутация) на IP адреса на източника.

Блокирането на имейл от изходния IP адрес ще изтече след един час. Ако IP адресът източник е локален имейл сървър, който ви принадлежи, проверете конфигурацията на конектора за пощенския поток. Ако поведението продължи повече от един час, обърнете се към поддръжката, за да поискате изключение за IP адреса на източника.
