# <a name="how-to-type-special-characters-on-a-mac"></a>Как се въвеждат специални знаци на Mac

За да въведете специални знаци на своя компютър Mac, отворете менюто **специални знаци** с клавишната комбинация на **командата**за  +  **управление**на ⌘  +  **Space**.

Бърза комбинация за @ е също и **Alt**  +  **64**.
