# <a name="how-to-type-special-characters-on-a-mac"></a>Как да въведете специални знаци на Mac

За да въведете специални знаци на вашия Mac, отворете менюто **специални знаци** с клавиатура къс ⌘ **команда** + **контрол** + **пространство**.

Бърз пряк път за @ също е **ALT** + **64**.
