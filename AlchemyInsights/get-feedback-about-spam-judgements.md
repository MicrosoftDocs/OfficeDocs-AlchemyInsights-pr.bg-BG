# <a name="new-feature-get-feedback-about-spam-judgments"></a>Нова функция: получите обратна информация за спам решения

Като администратор можете да сега лесно доклад подозрителни имейл съобщения, URL адреси и съдържание на Microsoft в [Изявления Explorer](https://protection.office.com/reportsubmission). Можете дори да го използвате за да разбере защо законен съобщение е маркиран неправилно. За повече информация вижте това [съобщение](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622).

За да направите това easer, можете да разрешите вашите потребители да докладва директно на Microsoft чрез [добавката към съобщението](https://appsource.microsoft.com/product/office/WA104381180?src=office&tab=Overview).
