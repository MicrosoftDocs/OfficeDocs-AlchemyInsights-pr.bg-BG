# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>Надстройване до пълната версия на Microsoft Teams

Трябва да надстроите от безплатната до пълната версия на Microsoft Teams? Вижте [надстройване на Microsoft Teams, безплатна версия на абонамент за microsoft 365](https://docs.microsoft.com/microsoftteams/upgrade-freemium)

Ще трябва да отговаряте на следните изисквания:

- Вие сте човекът, който се е регистрирал за текущата безплатна версия на Teams.
- Ако имате собствен домейн, той все още не може да е свързан с Azure Active Directory.
- За да прехвърлите данни, когато надстройвате, трябва да се абонирате по време на процеса на надстройка. Ако сте закупили абонамент извън процеса на надстройване, не можете да прехвърлите данни.
