# <a name="webpage-html-web-resources-limitations"></a>Ограничения за уеб ресурси на уеб страница (HTML)

* HTML уеб ресурс не може да съдържа кодове, които трябва да бъдат изпълнени на сървъра. Страниците на ASP.NET не могат да се качат като HTML уеб ресурси.

* HTML уеб ресурсите могат да приемат само ограничен брой параметри на низ на заявка. Още информация: [преминавате параметри към HTML уеб ресурси](https://docs.microsoft.com/dynamics365/customer-engagement/developer/webpage-html-web-resources#BKMK_PassingParametersToWebResources)

[Уеб ресурси на уеб страница (HTML)](https://docs.microsoft.com/dynamics365/customer-engagement/developer/webpage-html-web-resources)
