# <a name="pst-data-files-in-outlookcom"></a>PST файлове с данни в Outlook.com

Не е възможно да импортирате Outlook. PST файл с данни за вашия акаунт за Outlook.com с помощта на браузър.

За да импортирате. PST файл с данни, трябва да използвате настолното приложение Outlook. За да научите повече, прочетете [Импортиране на имейли, контакти и календар от. pst](https://support.office.com/article/431a8e9a-f99f-4d5f-ae48-ded54b3440ac?wt.mc_id=Office_Outlook_com_Alchemy) файл на Outlook.
