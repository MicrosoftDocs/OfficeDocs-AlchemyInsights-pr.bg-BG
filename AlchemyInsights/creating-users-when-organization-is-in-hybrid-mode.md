# <a name="create-users-in-hybrid-deployments"></a>Създаване на потребители в хибридно разполагане

Когато вашата организация е хибридно разполагане между локална среда и Microsoft 365, трябва да създадете всички обекти на потребителя във вашата локална Active Directory. Създавате обекти на потребителя в съответната организационна единица (ОЕ), така че те са синхронизирани във вашата организация.
