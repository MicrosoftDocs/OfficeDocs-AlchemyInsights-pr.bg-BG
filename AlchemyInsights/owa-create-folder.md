# <a name="create-a-folder"></a>Създаване на папка

1. В дъното на прозореца на папката изберете **нова папка**.
2. В полето нова папка въведете име за папката и натиснете клавиша Enter.

За да научите повече, вижте [работа с папки за съобщения в Outlook в мрежата](https://support.office.com/article/ae0f10d6-54e7-4f29-acd3-78cdc3fdcb9f).
