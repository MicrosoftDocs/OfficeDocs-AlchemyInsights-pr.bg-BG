# <a name="copy-and-paste-in-outlookcom"></a>Копирайте и поставете в Outlook.com

Там не са копие или паста бутони в Outlook.com, но можете да използвате клавишни комбинации:

**На компютър:** CTRL + C за копиране, CTRL + X, за да изрежете и CTRL + V за да поставите.

**На Mac:** CMD + C за копиране, CMD + X, за да изрежете и CMD + V за да поставите.
