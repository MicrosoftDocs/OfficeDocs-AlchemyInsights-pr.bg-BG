# <a name="copy-and-paste-in-outlookcom"></a>Копиране и поставяне в Outlook.com

Има бутони за копиране или поставяне в Outlook.com, но можете да използвате клавишни комбинации:

**На компютър PC:** Натиснете CTRL + C, за да копирате, CTRL + X, за да изрежете, и натиснете CTRL + V, за да поставите.

**На Mac:** CMD + C за копиране, CMD + X, за да изрежете, и CMD + V, за да поставите.
