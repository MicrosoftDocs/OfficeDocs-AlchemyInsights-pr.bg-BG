# <a name="create-an-allow-list-of-endpoints-for-microsoft-edge"></a>Създаване на списък с крайни точки за Microsoft Edge

За да имате пълния набор от функции на Microsoft Edge на ваше разположение, трябва да сте свързани към интернет. Освен това, за да позволите на браузъра да комуникира чрез защитни стени и други механизми за защита в интернет, трябва да добавите набор от URL адреси на домейни към неговия [списък с крайни точки](https://go.microsoft.com/fwlink/?linkid=2135054).
