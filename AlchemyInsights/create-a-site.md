# <a name="create-a-site"></a>Създаване на сайт

- Отидете на [страницата "сайтове" в центъра за администрация на Microsoft 365](https://portal.office.com/adminportal/home#/SitesList)а след това изберете **Добави сайт**. 
    
- В новата SharePoint администратор център, на страницата на сайтове щракнете върху **Създай**. 
    
 **Забележка:** Не можете да създадете нов сайт, който има същия URL със съществуващ сайт. 
  
За повече информация за създаването на сайт вижте [Създаване на сайта](https://go.microsoft.com/fwlink/?linkid=866295).
  
За информация за разрешаване на потребителите да създават свои собствени сайтове вижте [Създаване на управление сайт в SharePoint Online](https://go.microsoft.com/fwlink/?linkid=866296).
