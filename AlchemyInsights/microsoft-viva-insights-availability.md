# <a name="microsoft-viva-insights-availability"></a>Достъпност на Microsoft Viva Insights

Microsoft Viva Insights е наличен в момента в предварителния преглед.

[Изтегляне](https://aka.ms/InsightsDocumentation) Данни за Viva в Microsoft Teams.

[Научете повече](https://aka.ms/VivaInsights) за начина, по който Viva Insights ви помага да създадете култура, в която хората и бизнеса могат да процъфтяват.

За повече информация вижте как [Microsoft Viva Insights помага на хората да подхранват благосъстоянието и да бъдат най-добри](https://techcommunity.microsoft.com/t5/microsoft-viva-blog/microsoft-viva-insights-helps-people-nurture-wellbeing-and-be/ba-p/2107010).
