# <a name="sharepoint-publishing-infrastructure"></a>SharePoint публикуване инфраструктура


<p>SharePoint включва класически набор от функции, които ви позволяват да автор и публикува богат уеб страници за вашия интранет. Активиране на SharePoint публикуване инфраструктурата на ниво на колекция със сайтове за да ги включите.</p>  <p>За повече информация моля посетете линка по-долу.</p>  <ul>  <li><a href="https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518">Разрешаване на функции за публикуване</a></li>  <li><a href="https://support.office.com/en-us/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553">Функции за публикуване</a></li>  </ul>  <p>Бележки:</p>  <ul>  <li>Класически публикуване сайт шаблони включват функции за публикуване по подразбиране.</li>  <li>Класически публикуване функции не са налични на модерни сайтове. Модерните сайтове предоставят по-нова алтернатива по подразбиране.</li>  </ul>
