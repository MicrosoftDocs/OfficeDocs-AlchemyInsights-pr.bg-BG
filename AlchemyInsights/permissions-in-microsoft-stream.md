# <a name="permissions-in-microsoft-stream"></a>Разрешения в Microsoft Stream

По подразбиране, когато качвате видеоклип, разрешенията се задават за видеоклипа, който да бъде видим за цялата фирма. Ако искате да зададете разрешения за определени потребители/групи, можете да персонализирате разрешенията за видео.

- [Разрешения и поверителност в Microsoft Stream](https://docs.microsoft.com/stream/portal-permissions)

- [Разрешения на група](https://docs.microsoft.com/stream/portal-permissions#group-permissions)

- [Разрешения за канал](https://docs.microsoft.com/stream/portal-permissions#channel-permissions)
