# <a name="edit-a-read-only-file"></a>Редактиране на файл само за четене

В библиотеки с документи когато щракнете върху името на документа, програмата от Office може неочаквано го отворите като файл само за четене. За да го отворите за редактиране, щракнете върху **Отвори** (три точки) и след това щракнете върху **Редактиране**
