# <a name="edit-a-read-only-file"></a>Редактиране на файл само за четене

В библиотеки с документи, когато щракнете върху името на документ, програмата на Office може неочаквано да го отворите като файл само за четене. За да го отворите за редактиране, щракнете върху **Отвори** (трите точка) и след това щракнете върху **Редактиране.**
