# <a name="seeing-a-false-alert"></a>Виждане на фалшива тревога

- Изпратете файл на Microsoft за анализ. За повече информация вижте [Какво да правите с фалшиви положителни/отрицателни резултати в Microsoft Defender Antivirus.](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#submit-a-file-to-microsoft-for-analysis)
- Създайте индикатор "Позволи", за да предотвратите фалшиво положително предупреждение. За повече информация вижте [Създаване на индикатори за файлове](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/indicator-file).  
- Конфигуриране на изключванията на устройството, за да предотвратите сканирането на даден елемент. За повече информация вижте [Дефиниране на изключване на отделно устройство с Windows, за да предотвратите сканирането на даден елемент](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#define-an-exclusion-on-an-individual-windows-device-to-prevent-an-item-from-being-scanned).
