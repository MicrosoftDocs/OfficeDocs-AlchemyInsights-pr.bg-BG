# <a name="seeing-a-false-alert"></a>Виждане на фалшиво известие

- Подайте файла на Microsoft за анализ. За повече информация вижте [какво да правите с неверни позитиви/негативи в Microsoft Defender Antivirus](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#submit-a-file-to-microsoft-for-analysis).
- Създайте индикатор "позволи", за да предотвратите фалшиво положителен сигнал. За повече информация вижте [Създаване на индикатори за файлове](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/indicator-file).  
- Конфигуриране на изключвания на устройството за предотвратяване на сканирането на елемент. За повече информация вижте [Дефиниране на изключение на отделно устройство с Windows, за да се предотврати сканирането на даден елемент](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#define-an-exclusion-on-an-individual-windows-device-to-prevent-an-item-from-being-scanned).
