# <a name="dashboard-doesnt-reflect-changes-after-refresh"></a>Таблото не отразява промените след обновяването

Ако таблото не отразява промените след обновяването, изчакайте около 10-15 минути за обновяване, за да се отрази в плочките на таблото. Ако все още не се показва, Закачете отново визуализацията в таблото.

За повече информация вижте [отстраняване на неизправности при обновяване на сценарии](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios).
