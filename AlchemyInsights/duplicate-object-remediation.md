# <a name="duplicate-object-synchronization-errors"></a>Повтарящи се грешки при синхронизиране на обекти

Ако по време на синхронизация получавате една от следните грешки:

- Синхронизиран обект със същия адрес за прокси сървър вече съществува във вашия указател на Microsoft Online Services.

- Подобно съобщение за UserPrincipalName.

Използвайте новата диагностика на AADConnect, за да отстраните грешките при дублирани обекти. За повече информация вижте [диагностициране и възстановяване на дублирани грешки при синхронизиране на атрибут](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).
