# <a name="proxyaddress-incorrect"></a>ProxyAddress неправилен

Когато обект е синхронизиран с Azure AD, стойностите, които са зададени в атрибута proxyAddresses в Active Directory, се сравняват с правилата на Azure AD, а след това атрибутът proxyAddresses се попълва в Azure AD. Следователно стойностите на атрибута proxyAddresses за обекта в Active Directory може да не съвпадат със стойностите на атрибута proxyAddresses в Azure AD.
  
За да научите повече за това как се попълва proxyAddress, вижте [как атрибутът proxyAddress се попълва в AZURE ad](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
