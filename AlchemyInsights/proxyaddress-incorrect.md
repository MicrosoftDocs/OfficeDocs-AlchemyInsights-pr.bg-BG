# <a name="proxyaddress-incorrect"></a>Неправилен прокси адрес

Когато обект се синхронизира Azure AD, стойностите, които са зададени в атрибута proxyAddresses в Active Directory се сравняват с Azure AD правила и след това proxyAddresses атрибут се попълва в Azure AD. Затова стойностите на атрибута proxyAddresses за обект в Active Directory не може да бъде същото като стойностите на атрибута proxyAddresses в Azure AD.
  
За да научите повече за това как се попълва proxyaddress, вижте [как атрибутът proxyAddress се попълва в Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
