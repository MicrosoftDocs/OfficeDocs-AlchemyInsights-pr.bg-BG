# <a name="proxyaddress-incorrect"></a>Прокси адресът е неправилен

Когато обект се синхронизира Azure AD, стойностите, които са зададени в атрибута прокси сървър в Active Directory са сравнени с правилата на Azure AD и след това прокси сървър атрибут се попълва в Azure AD. Затова стойностите на атрибута прокси сървър за обекта в Active Directory може да не е същото като стойностите на атрибута прокси сървър в Azure AD.
  
За да научите повече за това как се попълва прокси сървър, вижте [как се попълва прокси адрес в AZURE ad](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
