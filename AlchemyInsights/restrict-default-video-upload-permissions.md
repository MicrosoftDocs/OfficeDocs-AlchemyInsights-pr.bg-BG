# <a name="restrict-default-video-upload-permissions"></a>Ограничаване на разрешенията за качване по подразбиране за видеоклипове

Когато потребителите качат видео в Microsoft Stream чрез портала за потока извън контекста на група или канал на група, квадратчето за отметка позволявай на всички във вашата фирма да преглеждат това видео е отметнато по подразбиране. Това прави видеоклипът достъпен за преглед от всеки във вашата фирма.

- [Задаване на разрешения за качване на видео по подразбиране](https://docs.microsoft.com/stream/default-video-permissions)
