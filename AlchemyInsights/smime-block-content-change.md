# <a name="outlook-will-now-default-block-external-content-in-smime"></a>Outlook ще сега по подразбиране блок външно съдържание в S/MIME

Сме за въвеждане на мерки за прибавка сигурност в Outlook за Windows от блокирането на външно съдържание в S/MIME шифровани и подписани съобщения по подразбиране и предотвратяване на предимство на съобщение основа.

За пълни подробности моля вижте [Outlook за Windows блокира външно съдържание в S/MIME съобщения](https://support.office.com/article/2d3a4af1-fe41-475f-a888-fc7b997d112e).
