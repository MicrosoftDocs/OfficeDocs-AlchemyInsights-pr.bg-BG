# <a name="outlook-will-now-default-block-external-content-in-smime"></a>Outlook сега ще блокира външно съдържание в S/MIME

Представяме ви добавени мерки за сигурност в Outlook за Windows, като блокираме външно съдържание и сме регистрирали съобщения по подразбиране и не допускаме замествания на базата на съобщения.

За пълните подробности вижте [Outlook за Windows блокира външно съдържание при S/MIME съобщения](https://support.office.com/article/2d3a4af1-fe41-475f-a888-fc7b997d112e).
