# <a name="desktop-app-for-sharepoint"></a>Настолно приложение за SharePoint

Вашата организация е въведена в програмата за лични визуализации на настолното приложение за SharePoint. В момента Microsoft не поддържа настолното приложение за SharePoint (личен предварителен преглед). За да получите помощ за настолното приложение за SharePoint, използвайте вашите партньори на Microsoft за лични предварителни визуализации.
