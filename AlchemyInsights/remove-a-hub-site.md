# <a name="remove-a-hub-site"></a>Премахване на сайт концентратор

Дерегистриране на сайт концентратор в центъра за администрация на нови SharePoint
  
1. В новия център за SharePoint администратор под **сайтове**, щракнете върху **активни обекти**. 
    
2. Изберете сайта, щракнете върху **сайт концентратор**и след това щракнете върху **регистрацията като сайт концентратор**. 
    
3. Щракнете върху **OK**. 
    
[Научете повече за концентратор сайтове](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f?ui=en-US&amp;rs=en-US&amp;ad=US)
