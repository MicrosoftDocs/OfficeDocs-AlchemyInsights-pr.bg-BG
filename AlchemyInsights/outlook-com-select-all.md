# <a name="how-to-select-all-in-outlookcom"></a>Как да "Избери всички" в Outlook.com

Можете да изберете всичко, само ако всички съобщения са видими. Това е да се предотврати случайно действия върху съобщенията, които не можете да видите.

**CTRL + щракване** избира множество отделни съобщения.

**SHIFT + кликнете** избира всички съобщения между първото и последното съобщение, изберете.
