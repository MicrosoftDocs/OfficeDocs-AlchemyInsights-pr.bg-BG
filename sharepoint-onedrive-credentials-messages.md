# <a name="credential-messages-in-sharepoint-or-onedrive"></a>Съобщения за идентификационни данни в сайт на SharePoint или OneDrive

<p>Потребителите могат да бъдат периодично питани от Службата за идентификационни данни на SharePoint Online или OneDrive при опит за отваряне на документи.</p> <p>По-долу са някои примери за идентификационни данни съобщения:</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Влезете в профила си организационни</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Идентификационни данни са задължителни</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Ние сме в състояние да се свържете сега, проверете вашата мрежова връзка.</p> <p>Ако имате тези симптоми, моля, изпълнете стъпките в <a href="https://support.microsoft.com/en-us/help/2913639/office-applications-periodically-prompt-for-credentials-to-sharepoint">тази статия</a>.</p> <p>Като алтернатива ако ти сте using единствен знак-на (SSO) за удостоверяване, моля вижте <a href="https://support.microsoft.com/en-us/help/4025962/cant-sign-in-after-update-to-office-2016-build-16-0-7967-on-windows-10">тази статия</a> вместо това.</p> <p>&nbsp;</p>
